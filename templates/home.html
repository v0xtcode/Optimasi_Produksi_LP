Tentang Proyek

Latar Belakang

Aplikasi ini dikembangkan sebagai bagian dari tugas mata kuliah Sistem Pendukung Keputusan di Universitas Hasyim Asyari. Aplikasi ini bertujuan untuk menerapkan konsep Linear Programming dalam konteks optimasi produksi yang relevan dengan dunia bisnis dan industri.

Tujuan

  • Mengimplementasikan metode Linear Programming dalam bentuk aplikasi web interaktif
  • Membantu perusahaan dalam mengoptimalkan perencanaan produksi
  • Memvisualisasikan hasil optimasi agar lebih mudah dipahami
  • Memberikan analisis mendalam tentang penggunaan sumber daya dan faktor pembatas

Metodologi

Aplikasi ini menggunakan pendekatan Linear Programming untuk menyelesaikan masalah optimasi produksi. Linear Programming adalah teknik matematis yang mencari solusi optimal dari fungsi tujuan linear dengan mempertimbangkan sejumlah batasan linear.

Dalam konteks aplikasi ini, fungsi tujuan adalah memaksimalkan keuntungan total, sementara batasan-batasannya adalah ketersediaan sumber daya seperti bahan baku dan tenaga kerja.
